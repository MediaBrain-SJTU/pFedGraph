# Personalized Federated Learning with Inferred Collaboration Graphs, ICML2023

**Note:** This repo is still in progress.

## Introduction
This repo is the official implementation of **Personalized Federated Learning with Inferred Collaboration Graphs** ([ICML Link](https://proceedings.mlr.press/v202/ye23b/ye23b.pdf)), which has been accepted by ICML, 2023.

We have open-sourced our codes on three libraries, including Pytorch, TensorFlow, PaddlePaddle, and MindSpore! Please refer to the corresponding sub-folder for detailed README.
